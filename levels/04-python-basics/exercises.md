# 第 04 关：加分与变式练习（`exercises.md`）

以下 4 项与 `starter` 中脚本对应，可在完成主线后再做。

---

## 练习 1：修改变量并运行

**目标：** 理解变量是「带名字的值」，改完后 `print` 输出会跟着变。

**操作步骤：**

1. 用编辑器打开 `01_print_and_variables.py`。  
2. 修改 `name`、`age`、`goal` 为你自己的内容（字符串用引号，数字不用引号）。  
3. 保存文件。  
4. 在终端进入 `starter` 目录，执行 `python 01_print_and_variables.py`（或 `python3` / `py`）。

**成功标志：** 终端里打印的文字与你的修改一致；若用了 f-string，句子读起来通顺。

**常见错误：**

- 字符串忘了加引号，运行报 `SyntaxError`。  
- 改了文件但没保存，终端里仍是旧输出。  
- 不在 `starter` 目录下运行，用了错误路径（本关脚本与 `README` 同级，直接 `python 01_…` 即可）。

## 练习 2：修改 list 和 dict

**目标：** 分清「列表是一串有序项」和「字典是键到值的映射」。

**操作步骤：**

1. 打开 `02_list_and_dict.py`。  
2. 在 `skills` 里增删或改掉某几项字符串。  
3. 在 `task` 字典里改 `status`、`progress`、`message` 的值（保持键名不变）。  
4. 保存后运行 `python 02_list_and_dict.py`。

**成功标志：** 列表循环打印的每一行、以及字典三行输出都反映你的修改。

**常见错误：**

- 列表或字典最后一项后面多写了逗号以外的语法错误（少括号、少引号）。  
- 把字典的键名打错，例如写成 `statu`，读取会得到 `KeyError`（本练习不要求处理异常，改回正确键名即可）。

---

## 练习 3：修改 if 判断条件并观察输出

**目标：** 知道 `if` 用来在「不同条件」下走不同分支。

**操作步骤：**

1. 打开 `03_if_and_loop.py`。  
2. 修改 `score` 或 `status`（按文件里注释提示），使判断结果从一种分支变到另一种。  
3. 修改 `items` 列表里的学习任务文字。  
4. 保存后运行 `python 03_if_and_loop.py`。

**成功标志：** 分数或状态变化时，「通过 / 需巩固」一类提示会跟着变；`for` 打印的清单与列表内容一致。

**常见错误：**

- 数值与字符串比较（例如 `score > "80"`），可能引发类型问题；请保持 `score` 为数字。  
- 缩进被打乱，`IndentationError`——`if` 下面缩进的块要对齐。

---

## 练习 4：修改函数参数并观察输出

**目标：** 理解「定义函数 → 传参数 → 用返回值」的最小闭环。

**操作步骤：**

1. 打开 `04_function.py`。  
2. 修改调用函数时传入的 `name` 或 `task_name`（按文件内说明）。  
3. 可选：微调函数里返回的字符串模板。  
4. 保存后运行 `python 04_function.py`。

**成功标志：** 终端打印的提示语里出现你传入的名字或任务名。

**常见错误：**

- 调用函数时括号或引号不配对。  
- 函数内 `return` 缩进不在函数体内部，导致 `SyntaxError`。
