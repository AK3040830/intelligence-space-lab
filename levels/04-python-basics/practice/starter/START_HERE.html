第 04 关：本地练习（Python 基础语法）

本页假设你已经完成第 01 关（会在终端里 cd、用 python 跑脚本）与第 02 关（会用编辑器打开文件夹、保存文件）。

本关不要求 Git、HTTP、安装第三方库。

只看网页不算完成。你必须在本地真的运行下面的 Python 文件，并至少修改并保存 2 个 .py 文件后再次运行，看到输出变化。然后再填写 notes/task-note.txt。
Windows 提示：若终端里输入 python 提示找不到命令，可尝试 py 01_print_and_variables.py 或 python3 …（视你本机安装而定）。

1. 打开 starter 文件夹

确认你在本关的 starter 目录里（来自 package.zip 解压后的 level-04-python-basics-starter/，或整仓里的 levels/04-python-basics/practice/starter/）。应能看到四个 .py 文件与 notes/。

2. 用编辑器打开整个 starter 文件夹

VS Code / Cursor：「文件」→「打开文件夹…」→ 选中整个 starter。左侧应列出 01_print_and_variables.py 等。

3. 打开终端，并进入 starter

  1. 打开终端（PowerShell、Terminal 等）。
  2. 使用 cd 进入本文件夹（与 START_HERE.html 同级）。可用 pwd（Linux/macOS）或 cd（Windows）确认当前路径。

4. 运行第一个脚本

  1. 执行：python 01_print_and_variables.py
  2. 在编辑器里修改 name、age、goal，保存后再运行一次，观察终端输出变化。

5. 列表与字典

  1. 运行：python 02_list_and_dict.py
  2. 修改 skills 列表与 task 字典里的 status / progress / message，保存后再运行。

6. 判断与循环

  1. 运行：python 03_if_and_loop.py
  2. 修改 score 或 status、以及 items 列表文字，保存后再运行，看 if 分支与 for 输出如何变化。

7. 简单函数

  1. 运行：python 04_function.py
  2. 修改传给函数的 student、next_task（或按文件内注释微调返回文字），保存后再运行。

8. 填写学习笔记

打开 notes/task-note.txt，在每一行提示后写你自己的话（不必很长），保存。

9. 对照 checklist 自检

在本关根目录（starter 的上一级再上一级）打开 checklist.md：路径为 levels/04-python-basics/checklist.md。若你只有练习包 ZIP，可向负责人索取同级的 checklist.md，或使用整仓材料对照。

建议提交物：终端运行截图、修改后的 Python 文件截图、notes/task-note.txt 截图、再加一句话总结。

完成后可在在线训练营进入第 05 关：HTTP / URL / 浏览器请求（规划中）。
