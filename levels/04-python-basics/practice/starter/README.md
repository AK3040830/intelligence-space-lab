# 第 04 关 · 练习包（starter）

本文件夹是本关最小练习集。请先阅读上级目录的 **`../demo/index.html`**（网页卡片），再在本文件夹双击 **`START_HERE.html`** 跟步骤做。

## 文件清单

| 文件 | 说明 |
|------|------|
| `START_HERE.html` | 本地练习主入口（推荐从这里开始） |
| `01_print_and_variables.py` | print、变量、字符串与数字 |
| `02_list_and_dict.py` | `list` 与 `dict` |
| `03_if_and_loop.py` | `if` 与 `for` |
| `04_function.py` | 定义与调用简单函数 |
| `notes/task-note.txt` | 学完后填写几句总结 |

## 工作目录

在终端运行脚本时，当前目录应是 **`starter`**（与本 `README.md` 同级），例如：

```bash
python 01_print_and_variables.py
```

若 `python` 不可用，可尝试 `python3`（常见于 macOS/Linux）或 Windows 的 **`py`**。

## 与在线播放器的关系

仓库内完整路径为：`levels/04-python-basics/practice/starter/`。下载 **`package.zip`** 解压后，顶层文件夹名为 **`level-04-python-basics-starter/`**。
