# 第 04 关：Python 基础语法与数据结构

## 1. 本关目标

用最少的语法点，让你**真的写过、跑过**一点点 Python，并建立这些词的直觉：

- `print` 输出、**变量**、**字符串**与**数字**
- **列表** `list`、**字典** `dict`
- **`if`** 判断、**`for`** 循环、**简单函数**
- 在终端里 **`python 某个文件.py`** 的运行方式（与第 01 关衔接）

**本关不是完整 Python 教程**，只为让你后面读 **HTTP、API、Django、脚本和任务状态** 里的示例代码时不完全陌生。

## 2. 为什么要学这一关

你已经会：打开文件、用编辑器保存、在终端里进对目录（第 01–02 关），也动过网页三件套（第 03 关）。接下来很多关会反复出现「用 Python 读数据、打印字段、改一个小脚本」——如果变量、`list` / `dict`、`if` / `for` 完全没概念，会直接卡在报错上。

本关用 **四个很小的 `.py` 文件** 让你改几个字、保存、再运行，把「语法名词」和「终端里看到的效果」对上号。

## 3. 在线学习入口

在仓库中打开（或用浏览器打开托管页后进入相对路径）：

- **`levels/04-python-basics/demo/index.html`** — 网页卡片教程与小测（不运行真实 Python，只建立直觉）

在线课程播放器中也可切换到 **第 04 关** 逐节阅读。

## 4. 本地练习入口

1. 下载本关 **`practice/package.zip`**（或从整仓进入 `levels/04-python-basics/practice/starter/`）。
2. 解压后进入顶层文件夹 **`level-04-python-basics-starter/`**（或仓库内的 `starter`）。
3. **双击**打开 **`START_HERE.html`**，按网页步骤操作。

**只看网页不算完成**：你必须在本地**至少运行过四个脚本中的若干个**，并**修改、保存至少 2 个 `.py` 文件**，再填写 `notes/task-note.txt`。

## 5. 你需要准备什么

- 已安装 **Python**（与第 01 关相同；若 `python` 不可用，可试 `python3` 或 Windows 的 **`py`**）。
- **编辑器**（VS Code / Cursor 等），会用「打开文件夹」打开整个 `starter`。
- **终端**，能 `cd` 到 `starter` 目录（第 01 关已练过）。

本关**不要求**：Git、HTTP、`requests`、第三方库、数据库、框架。
