第 05 关：本地练习（HTTP / URL / 浏览器请求）

本关假设你会用终端 cd、会用编辑器打开文件夹并保存（第 01、02、04 关）。

本关不要求 Git、curl、requests、安装第三方库。

本关重点不是双击 HTML。你必须用浏览器访问 http://127.0.0.1:8005/ 这类地址（本地 HTTP），才能体会「请求、响应、状态码」与 file:// 的差别。
端口与命令：本关统一使用端口 8005。启动命令为 python3 -m http.server 8005。若 python3 不可用，Windows 可试 py -m http.server 8005 或 python -m http.server 8005。

1. 打开 starter 文件夹

确认你在本关的 starter 目录中（ZIP 解压后为 level-05-http-url-browser-starter/，或整仓路径 levels/05-http-url-browser/practice/starter/）。应能看到 index.html、data.json 等。

2. 用编辑器打开整个 starter 文件夹

在 VS Code / Cursor 中选「打开文件夹」，选中整个 starter，左侧应列出本关文件。

3. 打开终端并进入 starter

在终端里 cd 到本文件夹（与 START_HERE.html 同级）。

4. 启动本地 HTTP 服务

  1. 执行：python3 -m http.server 8005
  2. 终端里应出现 “Serving HTTP on … port 8005” 一类提示。
  3. 保持该终端窗口不要关（关掉就等于停止服务）。

5～8. 用浏览器访问（新开浏览器标签即可）

  1. 打开 http://127.0.0.1:8005/（应看到首页 index.html）。
  2. 打开 http://127.0.0.1:8005/hello.html。
  3. 打开 http://127.0.0.1:8005/data.json。
  4. 打开 http://127.0.0.1:8005/not-exist.html，观察 404 或「未找到」提示；并阅读 missing-page-note.txt。

9～10. 修改文件并刷新

  1. 在编辑器中修改 index.html 的标题或说明文字，保存，回到浏览器 刷新 首页。
  2. 修改 data.json 里的 message 或 items 里一个词，保存，刷新 data.json 页面。

11. 填写 notes/task-note.txt

打开 notes/task-note.txt，在提示后写简短自己的话，保存。

12. 对照 checklist.md 自检

本关根目录（与 practice 同级）有 checklist.md。若只有 ZIP，可向负责人索取或与整仓材料对照。

建议提交物：终端里 http.server 截图、访问首页与 data.json 截图、404 截图、notes/task-note.txt 截图。

完成后可在在线训练营继续 第 06 关：JSON / API / curl / requests（规划中）。
