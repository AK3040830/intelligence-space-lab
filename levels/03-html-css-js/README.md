# 第 03 关：HTML / CSS / JavaScript 基础

## 1. 本关目标

让完全新人**第一次**理解：网页由 **HTML（结构）**、**CSS（样式）**、**JavaScript（交互）** 组成；并能在本地完成一个**最小**三件套练习：打开 → 修改 → **保存** → **刷新浏览器** 看到变化。

本关**不是**学习完整前端开发，只是先建立**网页三件套的第一层直觉**。

## 2. 为什么要学这一关

后面 Vue、HTTP、接口等都会假设你已经习惯「改前端文件、保存、刷新」。若这一关跳过，后面会在「不知道改哪个文件、为什么样式没变」上浪费大量时间。

## 3. 在线学习入口

- 卡片教程：`demo/index.html`（仓库路径：`levels/03-html-css-js/demo/index.html`）  
- 也可通过仓库根目录的在线训练营播放器进入本关小节。

## 4. 本地练习入口

1. 下载 **`practice/package.zip`**（或使用整仓 ZIP 进入 `levels/03-html-css-js/practice/starter/`）。  
2. 解压后进入 **`starter`**。  
3. 双击 **`START_HERE.html`** 跟步骤操作；练习主页为 **`index.html`**。

## 5. 你需要准备什么

- 浏览器、解压工具、**VS Code 或 Cursor**（推荐）  
- 已完成 **第 02 关** 的「打开文件夹、保存文件」基础（本关会反复使用）

**不需要：** Git、终端、Node、npm、Vue、Django、HTTP API。

## 6. 完成标准

- 已修改并保存 **`index.html`**、**`style.css`**、**`script.js`**，刷新后能看到结构与样式、按钮交互的变化  
- 已填写 **`notes/task-note.txt`**  
- 已对照 **`checklist.md`** 自检并准备提交物（听负责人安排）

## 7. 和后续课程的关系

第 04 关起进入 **Python 基础** 等路线；本关提供的「三件套分工 + 保存刷新」会在后续前端与全栈练习中反复出现。
