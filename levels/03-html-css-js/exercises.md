# 第 03 关 · 变式练习

在 `practice/starter/` 中完成。

---

## 练习 1：修改 index.html

**目标：** 分清「结构」由 HTML 负责；熟练改标题与正文并保存刷新。

**操作步骤：**

1. 用编辑器打开 `index.html`。  
2. 修改 `<title>`、`<h1>`、自我介绍段落、按钮文字。  
3. 保存，浏览器刷新 `index.html`。

**成功标志：** 浏览器里能看到你写的标题与介绍，按钮文字已变。

**常见错误：** 改错标签导致页面空白；忘记保存就刷新。

---

## 练习 2：修改 style.css

**目标：** 体会「样式」由 CSS 负责；改一处即可看到视觉变化。

**操作步骤：**

1. 打开 `style.css`。  
2. 修改 `body` 背景色或 `.card` 圆角或按钮颜色。  
3. 保存并刷新浏览器。

**成功标志：** 背景、圆角或按钮外观至少一项明显变化。

**常见错误：** 改错选择器；CSS 语法少了分号导致后面样式不生效。

---

## 练习 3：修改 script.js 并点击按钮验证

**目标：** 体会「交互」由 JavaScript 负责。

**操作步骤：**

1. 打开 `script.js`。  
2. 修改 `msg.textContent = "..."` 里的字符串为你自己的提示。  
3. 保存，刷新浏览器，点击按钮。

**成功标志：** 点击后显示你写的那句话。

**常见错误：** 引号不成对；保存了但浏览器仍缓存旧脚本（可强制刷新 `Ctrl+F5`）。
