第 03 关：HTML / CSS / JavaScript — 本地练习

本关不要求 Git、终端、Node、npm、Vue、Django、HTTP API。你只需要：资源管理器 + 浏览器 + 编辑器。

重要：只看网页、不在编辑器里保存任何修改，不算完成本关。你必须真的改 index.html、style.css、script.js，每次保存后回到浏览器刷新，并看到变化。

1. 准备 starter 文件夹

确认你在本关的 starter 目录里（来自 package.zip 解压后的 level-03-html-css-js-starter/，或整仓里的 levels/03-html-css-js/practice/starter/）。

2. 先看一眼网页

  1. 双击 index.html，用浏览器打开，记住标题、文字、按钮长什么样。

3. 用编辑器打开整个 starter 文件夹

在 VS Code / Cursor 中选 文件 → 打开文件夹，选中 starter。左侧应能看到 index.html、style.css、script.js、notes/。

不要只打开单个文件又不知道路径；整夹打开最稳。

4. 修改 index.html

  1. 改 <title> 里的页面标题。
  2. 改 <h1> 里的姓名或昵称。
  3. 改自我介绍那段 <p class="intro"> 里的文字。
  4. 改按钮上的文字（<button> 标签中间）。
  5. 保存（Ctrl+S / Cmd+S），回到浏览器对 index.html 按 F5 或 Ctrl+R 刷新，确认文字变了。

5. 修改 style.css

  1. 试着改 body 的 background 颜色。
  2. 改 .card 的 border-radius（圆角）。
  3. 可改按钮背景色或 #helloBtn 的 padding。
  4. 保存后刷新浏览器，看样式是否变化。

6. 修改 script.js

  1. 找到给 msg.textContent 赋值的那一长串中文。
  2. 改成你想在点击按钮后显示的一句话。
  3. 保存后刷新浏览器，点按钮，看文字是否按你写的变化。

7. 填写 notes/task-note.txt

打开 notes/task-note.txt，在每一问下面写至少一句自己的话，然后保存。

8. 对照 checklist.md

到本关根目录（与 practice 同级）打开 checklist.md，逐项自检并准备截图与一句话总结。
