# 第 03 关练习包（starter）

按 **`START_HERE.html`** 的顺序，用 **编辑器打开整个 `starter` 文件夹**，修改 **`index.html`**、**`style.css`**、**`script.js`**，并用浏览器打开 **`index.html`**，每次保存后 **刷新** 看效果。

| 文件 | 作用 |
|------|------|
| `index.html` | 网页结构：标题、段落、按钮；引入 CSS 与 JS |
| `style.css` | 网页样式：背景、卡片、按钮外观 |
| `script.js` | 网页交互：按钮点击后改文字 |
| `notes/task-note.txt` | 本关总结 |
| `START_HERE.html` | 操作步骤 |

上一级目录有 **`checklist.md`**、**`exercises.md`**。
