本页是网页卡片教程：只讲概念，不会在浏览器里真的运行 Python。动手跑脚本请到文件夹 practice/starter/，打开 START_HERE.html。

1. Python 是什么？

一种常用的编程语言。你可以用「人话」写逻辑，交给 Python 解释器执行。本关里我们会用一小段 Python 去读磁盘上的 JSON 文件并打印出来。

2. 终端是什么？

一个用文字输入命令的窗口（Windows 上常见 PowerShell /「终端」；Mac/Linux 叫「终端」）。后面运行 python demo/hello_task.py 就是在终端里输入的。

3. 当前目录是什么？

终端「此刻站在哪个文件夹里」。命令里的 demo/hello_task.py 是相对路径：相对当前目录去找 demo 文件夹。站错文件夹就会提示找不到文件。

4. JSON 是什么？

一种用大括号、英文双引号、逗号写出来的数据格式，常用来存「一堆键值对」。Python 读进来以后很像字典：可以用键名取出对应的值。

5. 为什么真正跑 Python 要放到练习包里？

浏览器里的卡片负责讲清楚概念；练习包里的 START_HERE.html 会一步一步写：开终端、进目录、输入哪条命令、出错了怎么办。这样你不会「光看网页却不知道怎么下手」。
