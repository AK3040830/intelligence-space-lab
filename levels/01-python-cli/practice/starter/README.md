# 第 01 关 · 练习包（starter）

本文件夹包含：

- **`START_HERE.html`** — 打开后按步骤在终端运行 Python（主学习路径）  
- **`demo/hello_task.py`** — 读取 `demo/task_status.json` 并打印字段  
- **`demo/task_status.json`** — 示例 JSON  
- **`verify.py`** — 验收脚本  

请先在上级目录阅读 **`../demo/index.html`**（网页卡片），再回本文件夹双击 **`START_HERE.html`**。

**工作目录：** 运行命令时，终端的当前目录必须是 **`practice/starter/`**（与本 `README.md` 同级）。
