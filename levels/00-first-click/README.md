# 第 00 关：第一次进入训练营

本关是 **Intelligence Space Lab** 的入口关，面向**完全零基础**新人。  
**不教** Python、任务状态字段、WebSocket 或调度器；只让你熟悉 **Lab 怎么学**：网页卡片 → 本地练习包 → 按步骤保存文件 → 提交截图。

---

## 这一关做什么

1. 知道 **Lab 新人训练仓** 是做什么的（闯关式学习材料，不是正式生产项目）。  
2. 分清 **网页卡片教程**（在浏览器里看）和 **本地练习包**（在你电脑上的文件夹里动手）。  
3. 会 **下载 ZIP / 找到文件夹 / 解压**（若需要）、**双击打开 HTML**、**打开并修改 `.txt` 文件**、**保存**、**截图**。  
4. 知道 **以后每一关** 的推荐顺序：**先看** `demo/index.html`，**再进** `practice/starter/`，**再打开** `START_HERE.html` 做练习。

---

## 为什么先学练习包和文件

后面很多关要在本机改代码、跑命令。若不会「打开正确文件夹、保存、刷新」，会直接卡在第一步。本关用 **纯文本 + 网页说明** 把动作练熟，**不要求安装任何开发环境**。

---

## 怎么打开网页卡片教程

1. 在资源管理器里进入：`levels/00-first-click/demo/`  
2. **双击** `index.html`，用浏览器打开。  
3. 依次点开页面上的 **四张卡片**，读展开后的说明。  
4. 阅读页面下方的「下载练习包」说明（本次未提供真实 `package.zip` 时，按提示直接进入 `practice/starter/` 即可）。

---

## 怎么进入练习包并完成练习

1. 进入文件夹：`levels/00-first-click/practice/starter/`  
2. **双击** `START_HERE.html`，按里面的步骤操作。  
3. 按 `START_HERE.html` 的指引打开并修改 `example-note.txt`，**保存**。  
4. 可选：完成同目录 `exercises.md` 里的小练习。

---

## 提交什么

见本关 **`checklist.md`**。通常包括：卡片页截图、完成练习后的截图、一句关于「练习包是什么」的说明。

---

## 和下一关的关系

完成本关后，进入 **`levels/01-python-cli/`**：那一关会在卡片里讲 Python/终端/JSON 概念，在 **`practice/starter/`** 里用命令行跑脚本。本关练好的「找文件夹、保存文件」会在下一关继续用到。
