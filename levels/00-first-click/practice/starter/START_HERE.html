第 00 关：练习包内部教程

你好！这个练习包不要求安装 Python、Node，也不要求打开终端。你只要会：找文件夹 → 打开文件 → 改几个字 → 保存 → 截图。

1. 这个练习包是干什么的？

让你熟悉「训练营」的学习方式：先看上一级目录里的网页卡片（demo/index.html），再到这个文件夹里动手改一个文本文件，证明你会保存。

2. 怎么解压？（如果你拿到的是 ZIP）

右键压缩包 → 选择「解压」或「全部提取」→ 选一个你找得到的位置（例如桌面）。解压后，按文件夹路径找到本关的 practice/starter/ 文件夹。

如果你是从整仓 ZIP 里打开的，只要能在资源管理器里进入 starter 文件夹即可，不必再解压一次。

3. 怎么找到文件？

你现在应能看到与本页面同目录下的：

  • START_HERE.html（本说明，可关可留）
  • README.md（简短文字说明）
  • example-note.txt（等下要改的练习文件）

4. 怎么打开 example-note.txt？

Windows：双击 example-note.txt，一般会「记事本」打开。

macOS：双击后可用「文本编辑」打开。

若已安装 VS Code：可以右键该文件 →「打开方式」→ VS Code。

5. 怎么修改文字？

在打开的文件里，用键盘把某一句话改成你自己的话（不要删光，保留两三行即可）。

6. 怎么保存？

Windows 记事本：菜单「文件」→「保存」，或按 Ctrl+S。

macOS：Cmd+S。

7. 怎么截图提交？

保存后，把窗口再打开一次确认内容已变，然后按你们系统的方式截图（例如 Windows Win+Shift+S），交给负责人或按群公告提交。

8. 可选：新建自己的笔记

在资源管理器空白处右键 →「新建」→「文本文档」，命名为 my-note.txt，写一句「我今天学会了什么」，保存。详见上一级目录的 exercises.md。
