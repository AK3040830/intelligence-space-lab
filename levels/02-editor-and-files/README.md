# 第 02 关：文件、编辑器与练习包使用

## 1. 本关目标

本关**不是**教编程语法，也不是 HTTP / Vue / Django。

本关只解决新人最容易卡住的基础操作：**认识文件与文件夹、ZIP 与解压、用编辑器打开整个练习文件夹、修改与保存、在浏览器里刷新看效果**，并理解 **`START_HERE.html`** 与 **`checklist.md`** 在训练营里的角色。

## 2. 为什么要学这一关

后面每一关都会让你在 **`practice/starter/`** 里改文件、看网页、对照清单验收。若这一关不扎实，后面会在「找不到文件」「没保存」「打不开文件夹」上反复消耗时间。

## 3. 在线学习入口

在浏览器中打开本关卡片教程（概念与小测）：

- `demo/index.html`（仓库内路径：`levels/02-editor-and-files/demo/index.html`）

也可通过仓库根目录在线训练营播放器进入本关对应小节。

## 4. 本地练习入口

1. 下载本关 **`practice/package.zip`**（或由整仓 ZIP 进入 `levels/02-editor-and-files/practice/starter/`）。  
2. 解压后进入 **`starter`** 文件夹。  
3. 双击 **`START_HERE.html`**，按网页里的步骤完成动手练习。

## 5. 你需要准备什么

- 能解压 ZIP 的文件管理器  
- 现代浏览器（Chrome / Edge / Firefox 等）  
- **VS Code** 或 **Cursor**（推荐；也可用其他编辑器，但步骤以 VS Code / Cursor 描述为主）

**不需要：** Git、终端、Python。

## 6. 完成标准

- 已按 `START_HERE.html` 修改并保存 **`hello-file.txt`**、**`profile-card.html`**、**`notes/task-note.txt`**  
- 已对照 **`checklist.md`** 自检通过  
- 已按清单准备截图与一句话总结（提交方式听负责人安排）

## 7. 和后续课程的关系

本关是 **第 03 关（HTML / CSS / JavaScript 基础）** 的地基：你会一直复用「打开文件夹 → 改前端文件 → 保存 → 刷新浏览器」这一套动作。
