第 02 关：本地练习（文件、编辑器、练习包）

本页不要求 Git、终端、Python。你只需要：资源管理器 + 浏览器 + 编辑器（VS Code 或 Cursor 等）。

重要：如果你只是用浏览器打开本页看一看，却没有在编辑器里保存任何练习文件，不算完成本关。请一定按下面步骤改文件并保存。

开始之前

  1. 确认你已经拿到本关的 starter 文件夹（来自本关 package.zip 解压后的 level-02-editor-and-files-starter/，或整仓 ZIP 里的 levels/02-editor-and-files/practice/starter/）。
  2. 建议安装 VS Code 或 Cursor，后面用「打开文件夹」最方便。

1. 打开 starter 文件夹

在资源管理器里进入本关的 starter 文件夹，你应该能看到 START_HERE.html、hello-file.txt、profile-card.html、notes 等。

2. 用编辑器「打开文件夹」（推荐）

打开 VS Code / Cursor → 菜单「文件」→「打开文件夹…」→ 选中整个 starter 文件夹（不要只选单个文件）。

只打开一个文件也能改，但很容易路径搞混；打开整个文件夹后，左侧文件树会一直陪着你。

3. 修改 hello-file.txt

  1. 在左侧文件树里点开 hello-file.txt。
  2. 把「我的名字是：______」那一行改成带你真名或昵称的一句话。
  3. 按 Ctrl+S（Windows）或 Cmd+S（macOS）保存。

4. 修改 profile-card.html

  1. 在同一编辑器里打开 profile-card.html。
  2. 把页面里的姓名、一句话介绍、以及按钮点击后提示文字（在 <script> 里）按注释改成你的内容。不要求你懂全部 HTML/JS，照抄改字即可。
  3. 保存文件。
  4. 用浏览器打开这个 profile-card.html（双击或在编辑器里右键用浏览器打开）。
  5. 点一下按钮，确认文字会变；若没变，回到编辑器检查是否保存，再刷新浏览器（F5 或 Ctrl+R / Cmd+R）。

5. 填写 notes/task-note.txt

打开 notes/task-note.txt，在三条提示下面各写一两句自己的话，然后保存。

6. 对照 checklist.md 自检

从 starter 往上看：先到 practice，再往外一层就是本关根目录，那里有 checklist.md（与 practice 文件夹同级）。逐项自检，确认你能用一句话说清 START_HERE.html 和 checklist.md 分别是干什么的。

7. 提交物（交给负责人）

  • START_HERE.html 在浏览器里打开的截图；
  • 修改后的 profile-card.html 在浏览器里的截图；
  • notes/task-note.txt 内容截图；
  • 再加一句话总结本关收获。

完成后可回到在线训练营播放器，继续第 03 关（HTML / CSS / JavaScript 基础，规划中）。
