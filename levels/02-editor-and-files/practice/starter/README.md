# 第 02 关练习包（starter）

本文件夹是**最小动手集合**。请按 **`START_HERE.html`** 里的顺序操作。

| 文件 / 文件夹 | 用途 |
|----------------|------|
| `START_HERE.html` | 主步骤说明（建议双击用浏览器打开） |
| `hello-file.txt` | 练习：改一行字并保存 |
| `profile-card.html` | 练习：改姓名、介绍与按钮提示，保存后在浏览器里刷新看效果 |
| `notes/task-note.txt` | 练习：写几句本关总结 |
| `README.md` | 本索引（你正在读） |

上一级目录有 **`checklist.md`** 与 **`exercises.md`**，用于自检与扩展练习。
